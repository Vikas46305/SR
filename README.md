## Student Management CRUD

This is a **Student Management System** built using **React.js** and **Redux Toolkit**.  
It is a simple CRUD application that allows users to manage student data.

## Features

- Add new students
- View student list
- Update student details
- Delete students
- Data persistence using Redux Persist

## Technologies Used

- React.js
- Redux Toolkit
- Redux Persist
- JavaScript
- CSS

## Installation

1. Clone the repository

```bash
git clone https://github.com/Vikas46305/SR
npm install
npm run dev
```
